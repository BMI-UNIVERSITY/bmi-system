## BMI University Web Application

Welcome to the official web application for BMI University. This repository contains the source code for the university's main website, built with modern web technologies to provide an engaging and seamless experience for students, faculty, and the public.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS (if configured) / Vanilla CSS
- **Language**: JavaScript (ES6+)

## 📦 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BMI-UNIVERSITY/bmi-university.git
   cd bmi-university
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application running.

## 📁 Project Structure

- `/app`: Contains the Next.js App Router pages and layouts.
- `/components`: Reusable UI components used across the application.
- `/public`: Static assets such as images and fonts.
